## SISTEMA DE CAIXA

### SOBRE O PROJETO
O repositório contém um sistema de caixa, criado como projeto da disciplina de "Algoritmos e Programação de Computadores".
Objetivo do projeto: aplicar os conhecimentos adquiridos na máteria de Algoritmo e Programação de Computadores, como criação de funções e controle de (I/O).

### FUNCIONALIDADES
- Cadastrar produtos
- Listar produtos
- Consultar produto por ID
- Remover produto
- Menu interativo

### TECNOLOGIAS ULTILIZADAS 
- Linguagem C
- Visual Studio COde
- GCC
  
### Pré-requisitos
- Compilador C (GCC)
